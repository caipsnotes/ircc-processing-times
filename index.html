About This Tool

This tool tracks IRCC (Immigration, Refugees and Citizenship Canada) processing times using official government data.

Data Sources

Data is automatically collected from official Canada.ca JSON endpoints:

  • Country names and codes
  • Current processing times
  • In-Canada services data

Updates

Current data is updated daily, and weekly snapshots are maintained for historical analysis.

GitHub Repository

This tool is maintained by immigration data enthusiasts to help track processing times.
